# LuxeCare Services Website

Production-style website for LuxeCare Services, a laundry and garment care business in Lagos, Nigeria.

## Stack

- React
- Vite
- Tailwind CSS
- Sanity CMS
- Netlify
- Cloudflare

## Features

- Multi-page responsive website
- WhatsApp booking
- Pickup request form using Netlify Forms
- CMS-ready services, pricing, FAQs, testimonials, and contact details
- SEO-ready structure with robots and sitemap
- Future-ready architecture for payments, accounts, and order tracking

## Content Rules

- Do not include first-time customer discount or 10% off messaging unless explicitly approved later.
- The flyer/PDF branding direction is a reference only. Do not copy any flyer layout exactly.
- Payment model at launch is pay on delivery.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site is intended to deploy automatically through Netlify when changes are pushed to GitHub.

- Build command: `npm run build`
- Publish directory: `dist`
